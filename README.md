## Project Overview
Automated unit testing framework for Java application with JUnit 5.

## Setup Instructions
1. Clone repository
2. Import as Maven project in Eclipse
3. Run: `mvn clean install`
4. Run tests: Right-click project → Run As → JUnit Test
